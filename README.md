colors
======

Flat UI based color scheme for MLVL visualizations.

Includes:

- ```highContrast```
- ```vis```
- ```blueShades```
- ```yellowShades```
- ```greenShades```
- ```orangeShades```
- ```purpleShades```
- [FlatUI](http://flatuicolors.co/) colors, by name.
